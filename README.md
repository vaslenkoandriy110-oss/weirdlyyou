# AI Knows Too Much — v6 Donatello Global Wall

Static viral personality website with:

- funny personality generator
- AADS ad slot
- Ukrainian-friendly donation buttons
- Donatello donor spotlight
- GitHub Actions sync for public donor messages

## How the real Donatello wall works

GitHub Pages cannot run backend code. This version uses a free workaround:

1. Visitor donates on Donatello and writes a message.
2. GitHub Actions runs every 5 minutes.
3. The action calls Donatello API using your private token.
4. It writes the latest messages into `donations.json`.
5. The website reads `donations.json` and shows messages in the live spotlight for 30 seconds.

This is not instant. Expect a delay of 5–10 minutes.

## Files

- `index.html` — site UI
- `styles.css` — design
- `app.js` — generator + donor spotlight logic
- `donations.json` — public donation feed generated by GitHub Actions
- `scripts/sync_donations.py` — Donatello API sync script
- `.github/workflows/sync-donatello-donations.yml` — GitHub Action
- `privacy.html` / `terms.html` — policy pages

## Step 1 — Put your real Donatello link into app.js

Open `app.js` and replace:

```js
donationUrl: "https://donatello.to/yourname",
donationLinks: {
  donatello: "https://donatello.to/yourname",
  monobank: "https://send.monobank.ua/jar/YOUR_JAR_ID",
  paypal: "https://www.paypal.com/paypalme/yourname",
},
```

with your real links.

Example:

```js
donationUrl: "https://donatello.to/weirdlyyou",
donationLinks: {
  donatello: "https://donatello.to/weirdlyyou",
  monobank: "https://send.monobank.ua/jar/abc123",
  paypal: "https://www.paypal.com/paypalme/weirdlyyou",
},
```

## Step 2 — Enable Donatello API and get token

In your Donatello account, find the API page, enable API functionality, and create an API token.

Do not paste the token into `app.js`. It must stay private.

## Step 3 — Add GitHub secret

In your GitHub repository:

1. Go to `Settings`.
2. Open `Secrets and variables`.
3. Open `Actions`.
4. Click `New repository secret`.
5. Name:

```text
DONATELLO_TOKEN
```

6. Value: paste your Donatello API token.
7. Save.

## Step 4 — Allow GitHub Actions to write donations.json

In your GitHub repository:

1. Go to `Settings`.
2. Open `Actions` → `General`.
3. Scroll to `Workflow permissions`.
4. Choose `Read and write permissions`.
5. Save.

The workflow also has:

```yaml
permissions:
  contents: write
```

## Step 5 — Run the sync manually once

In GitHub:

1. Open `Actions`.
2. Choose `Sync Donatello donations`.
3. Click `Run workflow`.
4. Wait for it to finish.

If it works, `donations.json` will update in your repository.

## Step 6 — Enable GitHub Pages

1. Go to `Settings` → `Pages`.
2. Source: `Deploy from a branch`.
3. Branch: `main`.
4. Folder: `/root`.
5. Save.

Your site will be available at:

```text
https://YOUR_USERNAME.github.io/YOUR_REPOSITORY/
```

## Important

- Donatello messages are synced through GitHub Actions, not directly from the browser.
- Never expose your Donatello token publicly.
- Do not ask users to click ads.
- Keep spam filtering strict because public donor messages can be abused.
